# arxiv-sanity-lite **Note**: The live site at https://arxiv-sanity-lite.com was not reachable during archiving (connection timed out). This may be temporary. ## Information - **Type**: Web application - Arxiv paper recommendation system - **Original Creator**: Andrej Karpathy - **Live URL**: https://arxiv-sanity-lite.com - **Source Repository**: https://github.com/karpathy/arxiv-sanity-preserver ## Description arxiv-sanity-lite is a web application that helps researchers keep up with the latest papers on arxiv. It provides a personalized recommendation system for arxiv papers, allowing users to: - Browse recent papers from various CS/AI subcategories - Like/dislike papers to train a personalized recommendation model - View papers recommended based on their preferences - Search and filter papers by keywords This was the successor/lite version of the original arxiv-sanity (now arxiv-sanity-preserver). ## Source Code The source code is available at: https://github.com/karpathy/arxiv-sanity-preserver
